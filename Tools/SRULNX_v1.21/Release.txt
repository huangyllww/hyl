ROM UPDATE - SRULNX

Description :
	Rom Update utility for Linux.

Release Versionï¼šv1.21
Release Date   : 2021.6.4

Usage:  
	SRULNX \<Input or Output File Name> [option]  
	or
	SRULNX \<Commands>
Commands:
	/D - Dump ROM ID.
	/V - Display BIOS Version.   
Options:                          
	/O - Dump SPI ROM's Binary. 
	/F - Force To Flashing BIOS Binary. 
	/R - Update ROM's Binary, And Then Shutdown System.
	/C - Update ROM's Binary, And Then Reset Tystem. 
	/N - Update ROM's Binary, And Do Not Program SMBIOS Interval.
	/P - Update ROM's Binary, And Do Not Program NVRAM Interval.
    /W - Update ROM's Binary, And Keep The Settings Password.
	/S - Verification Test Of Given ROM File Without Flashing BIOS.
Rules:
	1. Any parameter enclosed by < > is a mandatory field
	2. Any parameter enclosed by [ ] is an optional field
